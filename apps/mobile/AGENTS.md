# Expo SDK 54

Bu proje **Expo SDK 54**'te (telefondaki Expo Go bunu destekliyor; SDK 57'den
indirildi — bkz. memory: mobile-app-stack). Kod yazmadan önce sürüme uygun
dokümanı oku: https://docs.expo.dev/versions/v54.0.0/

Kurulumda `--legacy-peer-deps` şart (`.npmrc`'ye yazıldı). Sekmeler
`expo-router/ui` headless API ile (Tabs/TabList/TabSlot/TabTrigger).
